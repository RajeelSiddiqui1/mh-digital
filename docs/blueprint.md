# **App Name**: MH Enterprise Digital Media

## Core Features:

- Hero Section: Showcase headline, subtext, and CTAs with subtle background animation.
- Projects Section: Dynamically display project cards with images, titles, technologies, and links.
- Team Section: Showcase team members with profile images, roles, and hover effects.
- About Section: Display company overview, experience, and successful project count.
- Contact Section: Working form with name, email, message fields, company details, and Google Maps integration.
- Theme Toggle: Switch between light and dark themes.

## Style Guidelines:

- Light Theme: Background #F8FAFC, Text #0F172A, Primary #2563EB.
- Dark Theme: Background #0F172A, Text #F8FAFC, Accent #38BDF8.
- Font: 'Inter' (sans-serif) for headlines and body text. Note: currently only Google Fonts are supported.
- Use Lucide-react for icons.
- Fully responsive design for mobile, tablet, and desktop devices.
- Smooth transitions between sections using Framer Motion or AOS.
